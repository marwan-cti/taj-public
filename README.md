# TAJ Public — CTI & AI Security Notes

Public portfolio of short Cyber Threat Intelligence (CTI) notes under the TAJ project, focused on AI Security, malware, SOC, and threat hunting.

## Purpose

TAJ Public is the public-facing version of selected TAJ Factory notes. Each note is designed to show:

- a clear intelligence question;
- concise BLUF (Bottom Line Up Front);
- evidence-based judgments;
- source reliability using the Admiralty system;
- explicit uncertainty and limitations;
- practical defensive implications.

## Language

The main notes are currently written in French, with a short English summary for accessibility. Full English versions may be added later for selected notes.

## Methodology

The notes use a lightweight CTI structure:

1. Metadata
2. Executive summary / BLUF
3. English summary, when useful
4. Intelligence question
5. Key judgments
6. Observations
7. Defensive implications
8. Limitations
9. Analytical judgment
10. Sources

Source reliability is expressed with the Admiralty system:

- Source reliability: A-F
- Information reliability: 1-6

Example: `[B2]` means generally reliable source, information probably true.

## AI assistance disclosure

NotebookLM is used for source organization and citation extraction. ChatGPT is used for drafting support, structure review, critical review and editorial refinement.

Final analytical judgments, source reliability ratings, confidence levels and wording are reviewed and validated by the author.
